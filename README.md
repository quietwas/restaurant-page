# Restaurant Page

A restaurant website where the entire page content is generated dynamically using **JavaScript DOM manipulation** instead of static HTML.

## Overview

This project simulates a restaurant website with multiple sections such as **Home, Menu, and About**. Each section is rendered when the user clicks the corresponding navigation button.

## Features

- Dynamic page rendering with JavaScript
- Tab-based navigation (Home, Menu, About)
- Separate JavaScript modules for each page
- Content dynamically inserted into a single `#content` container
- Built and bundled using Webpack
